# AikoGx Center Obsidian

Repositorio de instrucciones para Codex sobre la nueva boveda de Obsidian de **Caos Entre Reinos: Reborn / Chaos Among Realms: Reborn**.

Regla publica: en documentacion, issues, README, prompts o archivos exportables usar solo **Fak**.

## Objetivo

Este repositorio explica como debe trabajar Codex con la boveda nueva:

- Migrar informacion desde la boveda antigua.
- Respetar la estructura visual creada por Fak.
- Convertir notas rapidas en tareas, ideas, devlogs y contenido util.
- Mantener compatibilidad con los widgets del Command Center.
- Evitar rehacer decisiones ya tomadas.

## Documentos

- `docs/GUIA_CODEX_MIGRACION.md`: instrucciones completas de migracion.
- `docs/WIDGETS_TAGS_PLANTILLAS.md`: tags, YAML y formatos de widgets.
- `docs/FLUJO_NOTAS_CODEX_Y_REDES.md`: flujo de Nota rapida para Codex, redes y calendario.
- `ideas-redes/PLANTILLA_IDEA_CONTENIDO.md`: plantilla para ideas de contenido.

## Principio base

```txt
Primero datos reales.
Luego widgets.
Luego estetica.
```

Codex no debe rehacer la arquitectura de la boveda. Fak decide la estructura; Codex migra, etiqueta, ordena y deja trazabilidad.
