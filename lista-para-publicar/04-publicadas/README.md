# Contenido publicado

Todavía no hay publicaciones registradas.

Cuando se publique una pieza, crear una ficha con:

- fecha de publicación;
- plataforma o web;
- URL pública;
- contenido y archivos utilizados;
- resultado conocido;
- comentarios o rendimiento;
- aprendizajes para mejorar;
- enlace al paquete original.

Nombre recomendado: `YYYY-MM-DD_nombre-corto_plataforma.md`.
