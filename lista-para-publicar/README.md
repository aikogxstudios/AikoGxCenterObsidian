## Estados

- **Listo para grabar:** guion y publicación preparados; falta capturar material.
- **Casi listo:** existe material base y falta edición o validación.
- **Lista para revisar:** contenido completo pendiente de aprobación de Fak.
- **Borrador:** todavía depende de decisiones importantes.
- **Publicado:** registrado dentro de `04-publicadas/`.

## Regla de mantenimiento

Cuando un contenido nuevo alcance utilidad real:

1. Mantener el documento completo en su carpeta de trabajo.
2. Añadir o actualizar su fila en este README.
3. Crear una ficha resumida en la subcarpeta correspondiente.
4. Al publicarlo, registrar fecha, plataforma, enlace y aprendizaje en `04-publicadas/`.
