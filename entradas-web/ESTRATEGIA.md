# Estrategia editorial web

## Objetivo

Preparar devlogs, artículos, textos SEO y publicaciones revisables para la web de **AikoGx Studios**, sin mezclar los artículos completos con las adaptaciones sociales.

## Contenido de esta carpeta

- borradores de entradas;
- devlogs listos para revisar;
- títulos SEO y alternativas;
- palabras clave;
- meta description y extracto;
- versión en español y, cuando proceda, en inglés;
- imagen destacada sugerida;
- propuestas para itch.io.

## Separación

- `entradas-web/`: artículos y devlogs completos.
- `ideas-redes/`: hooks, clips propuestos y adaptaciones sociales.
- Material Visual: capturas, clips, vídeos, referencias y assets reales.

## Voz editorial

Cercana, natural, indie, clara y agradable. No inventar avances, usar datos personales, prometer contenido inexistente ni sonar corporativo.

## Investigación

Antes de una entrada importante, revisar ejemplos actuales de devlogs indie, Unreal Engine, itch.io y juegos relacionados. Aprender estructura, SEO y presentación sin copiar.

Si no se puede investigar, incluir `Investigación pendiente`.

## Flujo

1. Confirmar los avances reales.
2. Definir objetivo y público.
3. Investigar referencias y palabras clave.
4. Redactar sin exageraciones.
5. Proponer imagen, slug, tags y CTA.
6. Crear por separado cualquier adaptación social dentro de `ideas-redes/`.
7. Dejar el estado en Borrador o Lista para revisar; no publicar automáticamente.
