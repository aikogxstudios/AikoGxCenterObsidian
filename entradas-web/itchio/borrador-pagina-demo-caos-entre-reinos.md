## Descripción larga en español

**Caos Entre Reinos: Reborn** es un proyecto indie de fantasía y acción desarrollado con Unreal Engine.

La aventura comienza con una cinemática sobre el equilibrio perdido entre los reinos y el despertar de una nueva esperanza. Después, el jugador toma el control dentro del Abismo: un espacio que conectaba mundos antes de la guerra y que ahora sirve como comienzo de su viaje.

El tutorial introduce movimiento, salto, dash aéreo, estilos de lucha, cofres y el sistema Orialis. Estas cartas forman la **Constelación** del jugador y permiten modificar aspectos como vida, daño y barra de caos.

La base actual de Orialis cuenta con 20 cartas integradas y tres slots funcionales. El proyecto también prepara la transición hacia la Aguja del Caos, una zona con ambición procedural, enemigos, cofres y diferentes tipos de habitaciones.

### Estado actual

La cinemática dispone de versiones en español e inglés. El tutorial tiene su flujo definido y sus diálogos organizados por zonas, pero todavía necesita repaso final, traducción adicional y pruebas.

La build `0.0.5.8` está planteada para una prueba privada con tres testers. No debe presentarse como una demo pública ni como una versión final optimizada.

## Long description in English

**Caos Entre Reinos: Reborn** is an indie fantasy action project developed with Unreal Engine.

The journey begins with a cinematic about the lost balance between realms and the awakening of a new hope. The player then takes control inside the Abyss, a place that once connected worlds before the war.

The tutorial introduces movement, jumping, aerial dash, fighting styles, chests, and the Orialis system. These cards form the player's **Constellation** and can modify health, damage, and the chaos gauge.

The current Orialis foundation includes 20 integrated cards and three functional slots. The project is also preparing the transition toward the Chaos Spire, an area planned around enemies, chests, and procedural room ideas.

### Current state

Spanish and English cinematic versions exist. The tutorial flow and dialogue zones are defined, but final polish, additional translation, and testing are still pending.

Build `0.0.5.8` is intended for a private test with three selected testers. It must not be presented as a public demo or a fully optimized release.
