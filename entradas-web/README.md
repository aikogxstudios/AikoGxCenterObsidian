# Entradas web

Carpeta oficial para devlogs, artículos, textos SEO y publicaciones web de **AikoGx Studios**.

## Estructura

| Carpeta | Uso |
| --- | --- |
| [`_plantillas/`](./_plantillas/) | Plantillas para entradas web e itch.io |
| [`00-estrategia/`](./00-estrategia/) | Estrategia editorial, SEO, tono y referencias |
| [`01-borradores/`](./01-borradores/) | Entradas todavía en escritura |
| [`02-en-revision/`](./02-en-revision/) | Borradores avanzados pendientes de revisión de Fak |
| [`03-listas-para-publicar/`](./03-listas-para-publicar/) | Entradas terminadas y preparadas para publicar |

## Requisitos antes de publicar

Una entrada lista debe incluir:

- título y slug;
- meta description;
- extracto;
- imagen destacada sugerida;
- tags;
- CTA;
- versión corta para redes;
- fuentes reales y estado confirmado.

## Reglas

- No inventar avances, fechas ni promesas.
- Mantener las adaptaciones sociales en `ideas-redes/`.
- Mantener capturas, clips y assets en Material Visual.
- No publicar automáticamente.
- Registrar URL, fecha y publicaciones derivadas al mover una entrada a `04-publicadas/`.

Consulta [la estrategia editorial y SEO](./00-estrategia/estrategia-web-seo.md) antes de preparar una entrada importante.
