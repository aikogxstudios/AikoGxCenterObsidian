# Entradas web

Carpeta para devlogs y entradas completas que estén preparadas para revisión o publicación web.

## Separación

- `entradas-web/`: artículos, devlogs y entradas web completas.
- `ideas-redes/`: adaptaciones sociales, hooks, clips propuestos y guiones cortos.
- `Caos Entre Reinos/11 - Material Visual/`: capturas, clips, vídeos, referencias y assets reales.

Una entrada web puede originar varias ideas sociales, pero cada adaptación debe guardarse como archivo independiente dentro de `ideas-redes/`.
